## Домашняя автосортировка 

### Проблема
- **Законодательное давление:** В развитых странах Азии (Япония, Южная Корея, Сингапур, Китай) вводятся строгие законы об обязательной сортировке отходов. За нарушения — крупные штрафы и общественное порицание.
- **Затраты времени:** Ежедневная ручная сортировка отнимает 5–10 минут в день → около **60 часов в год**.
- **Гигиенический дискомфорт:** Необходимость мыть грязную упаковку, контакт с пищевыми отходами, неприятные запахи.
- **Сложность правил:** Постоянно меняющиеся требования к типам пластика, бумаги, компоста.
- **Дефицит пространства:** Для нескольких контейнеров требуется дополнительное место на кухне.

### Решение
**«Домашняя автосортировка»** — умная домашняя станция для автоматической сортировки мусора.

**Как это работает:**
1. Пользователь бросает весь несортированный мусор в общий бак устройства.
2. Система анализирует отходы с помощью комбинации датчиков:
   - камера + ИИ для распознавания объектов;
   - (опционально) спектрометр для точного определения материалов.
3. Конвейерная система распределяет мусор по внутренним отсекам:
   - пластик, бумага, стекло, металл, общие отходы.
4. На выходе — аккуратные отсортированные пакеты, готовые к выносу в соответствующие контейнеры.

### Ключевые преимущества
- **0 времени на сортировку** — всё делает автомат.
- **0 контакта с грязными отходами** — полная гигиена.
- **Экономия пространства** — одно устройство вместо 4–5 контейнеров.
- **Интеграция в умный дом** — уведомления о заполнении, статистика по отходам.

## Полноценный AI-помощник 

### Проблема
- **Фрагментация экосистем:** Умные колонки, умный дом, компьютер — всё существует отдельно. Нет единого бесшовного интерфейса.
- **Пассивность ассистентов:** Голосовые помощники реагируют только на команды, не понимают контекст, не выполняют сложные многошаговые задачи на ПК.
- **Отсутствие «физического присутствия»:** Колонка — просто коробка. Робот с камерой и экраном может стать живым компаньоном.
- **Высокая цена и закрытость:** Существующие роботы (Amazon Astro, Sony Aibo) дороги и закрыты для кастомизации.
- **Сложность автоматизации:** Настройка сценариев в умном доме требует технических знаний.

### Решение
**Аппаратная часть:**
1. **Готовый робот** — полностью собранное устройство для массового потребителя.
2. **DIY-комплект** — набор деталей (корпус, шасси, сенсоры, микрокомпьютер) для энтузиастов, студентов, разработчиков. Подробная инструкция и ПО в комплекте.

**Программная часть (ядро продукта):**
- **Локальный AI-мозг** — основная обработка голоса и команд на устройстве (безопасность, скорость).
- **Интеграция с облачными AI-сервисами** (GPT, Claude, отечественные аналоги) для сложных задач.
- **Универсальные плагины/навыки:**
  - *Умный дом:* хаб для Zigbee, Matter, Wi-Fi.
  - *Компьютерный ассистент:* API для Windows/macOS/Linux (работа с файлами, браузером, офисом).
  - *Медиацентр:* управление ТВ, музыкой.
  - *Маркетплейс навыков* — для сторонних разработчиков.

### Конкуренты

| Категория | Примеры | Слабые стороны (наши возможности) |
|-----------|---------|-----------------------------------|
| **Голосовые ассистенты** | Алиса, Маруся, Siri, Google Assistant | Привязаны к брендам, нет физического носителя, ограничены в автоматизации ПК. |
| **Роботы-компаньоны** | Amazon Asto, Sony Aibo, Ubtech | Очень дорогие (Astro от $1600), закрытые экосистемы, нельзя собрать самому. |
| **DIY-робоплатформы** | Проекты на Arduino/Raspberry Pi | Требуют глубоких технических знаний, нет готовой отполированной программной части. |
| **Софт для автоматизации ПК** | Zapier, IFTTT, Power Automate | Работают по принципу «если-то», не на естественном языке, нет интеграции с физическим миром. |
| **Нишевые ассистенты** | Home Assistant | Акцент на приватность, но нет готового роботизированного аппаратного решения «под ключ». |
